Help

Residents

If you are a resident and need assistance:

  1. Register for an account using the Register link.
  2. Login and submit an aid request under "Request Aid".
  3. Check the "My Aid Requests" page to see your request status.
Barangay Staff

If you are barangay staff, you can review incoming requests and record distributions through the Admin dashboard.

  • Use filters to find requests quickly.
  • Record distributions after approval to update inventory.
